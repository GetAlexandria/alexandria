# Damien resources

Optional, load-on-demand context for the Damien agent (`../damien.md`). These
are kept out of the agent prompt to keep it token-light; Damien loads a file
only when the work calls for it.

| File | Load when… |
|---|---|
| `sociotechnica.md` | Doing strategy, positioning, audience, or voice work — who we are, the founders' roles, the goal, the values. |
| `campaign-strategy.md` | Deciding what a piece of media is *for* — the resourcing pyramid (acquirers → investors → clients), the per-audience pitch, the timeline, and what the media must do. |
| `new-media-role-and-playbook.md` | Designing the media system or answering org/process questions — the EP remit, the full New Media team's responsibilities, operating principles, process, governance, and KPIs. |
| `new-media-case-studies.md` | Wanting patterns/examples of how frontier operators (Every, a16z, HubSpot, Canva, Duolingo, YouTube, others) do this. Rarely needed day to day. |

The New Media docs were distilled from an external research report (citation
markers removed). The SocioTechnica and campaign docs come from internal
founder context.
